# 📋 Proje Hakkında

EY-CRM, Python ve PyQt5 ile geliştirilmiş, modern SaaS tarzı bir Müşteri İlişkileri Yönetimi (CRM) uygulamasıdır. Koyu tema, animasyonlu arayüz ve grafiksel raporlama özellikleri ile işletmelerin müşteri, satış ve destek süreçlerini tek platformdan yönetmesini sağlar.

# ✨ Özellikler

# 📊 Dashboard (Ana Panel)
- Canlı istatistik kartları (Toplam Gelir, Müşteri Sayısı, Satışlar, Açık Talepler)
- Aylık gelir grafiği (Bar Chart)
- Ürün bazlı gelir dağılımı (Pie Chart)
- Son aktivite akışı
- Otomatik yenileme (5 saniye)

# 👥 Müşteri Yönetimi
- Müşteri ekleme, düzenleme, silme
- Gelişmiş arama ve filtreleme
- Müşteri durumu takibi (Aktif/Pasif/Beklemede)
- Toplam harcama ve destek talebi istatistikleri
