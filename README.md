# Basit CRM Sistemi

Python ile geliştirilmiş terminal tabanlı CRM (Customer Relationship Management) uygulamasıdır.  
Müşteri, satış ve destek yönetimini temel seviyede sağlar.


# Özellikler

# Müşteri Modülü
- Müşteri ekleme
- Müşteri silme
- Müşteri listeleme

# Satış Modülü
- Satış ekleme
- Satış silme
- Satış listeleme
- Satış ID ile sorgulama

# Destek Modülü
- Destek talebi oluşturma
- Destek taleplerini listeleme
- Talep ID ile arama

# Kullanılan Yapılar

- Python sınıfları (OOP)
- Dictionary veri yapısı (müşteri ve satış kayıtları)
- List veri yapısı (destek talepleri)
- Terminal tabanlı kullanıcı arayüzü
